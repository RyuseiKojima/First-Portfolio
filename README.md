# 概要

ポートフォリオとして、"Shared kill time"というWebサイトを開発しました。
各ユーザーのお気に入り動画をシェアするサイトです。

# 背景

私は、今年の5月にKredo Online Campに入校し、以降アプリ開発の学習を継続しています。
Kredoは全授業が英語で実施され、このサイトはその一環として制作し始めたため、英語表記となっています。
